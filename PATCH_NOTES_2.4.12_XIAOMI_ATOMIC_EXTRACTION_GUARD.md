# NekoFlash 2.4.12 — Xiaomi ROM atomic extraction guard

## Цель

Убрать риск использования старых или частично извлечённых образов при повторной подготовке Xiaomi Fastboot ROM.

## Изменения

- Версия обновлена до `2.4.12-nekoflash`, `versionCode 45`.
- Перед распаковкой выбранных образов очищается рабочая папка конкретного ROM.
- Каждый образ извлекается во временный файл `*.partial`.
- После записи проверяется размер временного файла относительно размера entry в архиве.
- Только после успешной проверки временный файл заменяет финальный `.img` / `.bin` / `.elf` / `.mbn`.
- При несовпадении размера подготовка ROM останавливается до выполнения fastboot-команд.

## Почему это важно

Ранее рабочая папка уже имела fingerprint ROM, но существующий файл того же размера мог быть пропущен. Теперь подготовка не использует stale-файлы и не оставляет финальный образ, если извлечение оборвалось.

## Проверка

`scripts/check_project.py` должен проходить без ошибок.
