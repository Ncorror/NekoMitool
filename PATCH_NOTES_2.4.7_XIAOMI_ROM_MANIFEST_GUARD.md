# NekoFlash 2.4.7 — Xiaomi ROM image manifest guard

## Цель

Усилить Xiaomi Fastboot ROM Flasher перед реальной прошивкой: приложение теперь проверяет состав ROM-архива и referenced images из выбранного `flash_all`-сценария до выполнения fastboot-команд.

## Что добавлено

- ROM image manifest для `.zip`, `.tgz`, `.tar.gz`, `.tar` и распакованных папок.
- Отчёт по image-like entries: `.img`, `.elf`, `.mbn`, `.bin`.
- Проверка нулевых образов.
- Предупреждения о подозрительно маленьких образах меньше 512 B.
- Поиск дублирующихся имён образов в разных папках архива.
- Сверка образов, на которые ссылается выбранный `flash_all`-скрипт, с manifest ROM.
- Блокировка прошивки, если referenced image имеет размер 0 B.
- Дополнительная проверка уже извлечённых образов перед выполнением плана.

## Поведение

- Отсутствующий image по-прежнему блокируется на этапе подготовки плана.
- Zero-byte image теперь блокируется раньше, до flash.
- Маленький image не блокируется автоматически, потому что маленькие `vbmeta*.img` могут быть нормальными, но выводится предупреждение.
- Дубликаты basenames не блокируют ROM автоматически, так как реальные Xiaomi ROM могут хранить файлы в вложенной папке, но это отражается в dry-run отчёте.
