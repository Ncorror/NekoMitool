# UI redesign notes

Изменения:

1. Центральная зона больше не делится на маленькое окно + постоянный лог.
2. Разделы стали полноценными экранами внутри основного viewport:
   - Главная
   - Прошивка
   - Sideload
   - Файлы
   - Сервис
   - Консоль
3. Консоль и поле команд перенесены в отдельный экран `Консоль`, поэтому они не съедают место на всех остальных экранах.
4. Верхняя панель стала компактной: название, язык, статус устройства, поиск USB и стоп.
5. Нижняя навигация переключает целые окна, а не маленькую вкладку внутри прокручиваемого блока.
6. Исправлено поведение `currentTab`: экран `Файлы`, `Сервис` или `Консоль` больше не сбрасывает контекст ручной команды в Fastboot. Контекст меняется только при выборе Fastboot или ADB/Sideload.

Проверка в этой среде:

- XML успешно разобран как корректный XML.
- Полная сборка Gradle не выполнена, потому что wrapper попытался скачать Gradle с `services.gradle.org`, а в sandbox нет интернет-доступа.
